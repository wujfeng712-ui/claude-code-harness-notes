# s05 · TodoWrite — the planning tool

> **Motto: an agent with no plan goes wherever the wind blows.**

## What this lesson solves

A model is easily pulled off course by immediate tool feedback, drifting until it forgets what it set out to do. s05 adds `todo_write`: it performs no real action and adds not execution power but **planning power** — it forces the model to walk through the steps before acting, and pulls it back to its original intent when it drifts.

## How the mechanism works

- `todo_write` writes the plan into the in-memory `CURRENT_TODOS`; it has no side effects of its own.
- It pairs with a nag reminder: a `rounds_since_todo` counter that, once `>= 3` (three rounds with no TODO update), auto-injects a reminder so the model doesn't forget its original intent.

```
run a tool → rounds_since_todo += 1
              │
              ▼
       >= 3 ? ──yes──► inject "time to update your TODO" reminder
              │
              no → continue
```

## Key insights

- The value of a planning tool isn't in "what it did" but in "forcing the model to think before acting."
- Immediate feedback makes the model short-sighted; a three-round nag is a cheap course-correction.
